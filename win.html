How to run Windows™ 95© on the web

Setup:

  • Download and install dosbox to C:\dosbox
  • Download 622C and extract it into C:\dosbox
  • Download and install bochs to C:\bochs
  • Download and install OSFMount
  • Download and install 7-zip
  • Download the windows 95 installation and extract it to C:\windows95 using 7-zip
    •

    Steps:

    1. Copy C:\bochs\bximage.exe into C:\dosbox and run it, in the prompt press 1, and leave all the values as default (just press enter) except the hard disk size in megabytes which should be changed to 400
    2. Run C:\dosbox\DOSBox.exe and in it run:
    imgmount 2 c.img -size 512,63,16,812 -t hdd -fs none
    boot 622c.img
    fdisk
    1
    1
    Y
    1. When you see another prompt just press Enter to close DOSBox, then, open C:\dosbox\DOSBox.exe again and run:
    imgmount c c.img
    boot 622c.img
    format c:
    y
    win95
    1. Close DOSBox, then, extract the content of all the image files in C:\windows95 (except Boot.img) into a new folder named C:\win95 (extract the image files themself by opening them in 7-zip)
    2. Open OSFMount and choose Mount new… you need to find the image disk (C:\dosbox\c.img) then choose Mount all partitions option and uncheck the read-only option, then press ok, now if you open the file explorer you will see a new drive (D: in my case), drag&drop the C:\win95 folder into the new drive, then in OSFMount click on the new drive and then on Dismount
    3. Open C:\dosbox\DOSBox.exe and run:
    imgmount c c.img
    boot 622c.img
    mouse.com
    c:
    cd win95
    setup /is
    1. Open bash command line in the C:\dosbox folder (you might need to install it, you can follow Chris Hoffman’s guide) and run:
    sudo apt-get update
    yes | sudo apt-get install python python3 git binutils cmake automake
    git clone https://github.com/donno2048/em-dosbox
